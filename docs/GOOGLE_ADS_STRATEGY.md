# WebAdish Google Ads Optimization Strategy

This document outlines the negative keyword strategy and bidding focus for WebAdish high-ticket WordPress security services (£1,499+ / ₹25,000+).

## 1. Primary Objective
Exclude "DIY" hobbyists and low-budget users to focus budget on business-critical "Emergency Recovery" and "Enterprise Protection" leads.

## 2. Negative Keyword Lists
*Copy and paste these into your Google Ads "Negative Keyword Lists" section.*

## 3. Recommended Positive Keywords
Focus on **Urgency** and **Business Impact**.

| Category | Keywords |
|----------|----------|
| **Emergency** | `emergency wordpress recovery`, `hacked website fix uk`, `malware removal service india`, `wordpress security incident response` |
| **Recovery** | `restore hacked wordpress site`, `remove google blacklist`, `site suspended malware fix` |
| **Professional** | `wordpress security agency`, `enterprise wordpress protection`, `wp security retainer` |

## 4. Bidding & Geographic Strategy
- **UK:** Target major business hubs (London, Manchester, Birmingham) with a focus on "Emergency" recovery.
- **India:** Target Tier 1 cities (Mumbai, Bangalore, Delhi, Hyderabad) where business site downtime has high financial impact.
- **Bidding:** Use "Maximize Conversions" only after you have 15-30 conversions. Start with "Manual CPC" or "Enhanced CPC" to control spend on high-intent keywords.
